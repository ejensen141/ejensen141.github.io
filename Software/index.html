Software:

Open is a command for linux that imitates the Apple command open.

Physics Constants is a C header file that includes many physical constants for ease of use in Scientific Computing.

QPlotSci is a library for plotting scientific data to a QGraphicsView...easy to use with QtCreator as well as standalone.

EasyAVR is old software, no longer maintained.. although I may update it at some point. This is a GUI for compiling and uploading code to AVR devices.

ClaeLWS A Website/Blog application for building awesome websites.... not yet released Open Source.

Terminal Tools is a collection of tools that I use to make my coding life easier. These are written in bash. You can google, check gmail, run detached programs, and more right from the terminal with these easy commands.

d2a is a tool that takes ASCII data and converts that data to an audio file... Future versions may allow Binary data as well. I use this to make audio files from simulation or seismic data. Listening to data (waves) is somtimes an easier way of "visualizing" the data.
